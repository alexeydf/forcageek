Forca Geek

Instruções:

  • Se a palavra tiver "-", esse caractere será desconsiderado.
  • Para cada letra certa é ganho 1 ponto.
  • Para cada letra errada é perdido 1 ponto(A pontuação pode ficar negativa).
  • Acertando a palavra é ganho 10 pontos.
  • Caso seja enforcado os pontos são zerados.
